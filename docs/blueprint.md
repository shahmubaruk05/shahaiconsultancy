# **App Name**: BizSpark

## Core Features:

- User Authentication: Secure signup and login functionality.
- Business Strategy Generator: Generates a business strategy based on user inputs (business model, USP, pricing, marketing channels, and a 90-day action plan), then saves the generated strategy to Firestore.
- Startup Idea Validator: Validates startup ideas based on user input, returning a score, summary, risks, and recommendations, then saves these to Firestore.
- Pitch Deck Assistant: Generates a pitch deck outline based on user inputs. The tool provides slide suggestions, and saves the resulting pitch deck outline (an array of objects representing pitch deck slides) to Firestore.
- Company Formation Guide: Provides a company formation guide for Bangladesh and the USA (formation steps, document checklist, notes), and saves these to Firestore.
- Ask Shah Chatbot: An AI-powered chatbot assistant, that provides advice Startup, funding, licensing, tax, strategy, business and marketing . User messages are saved in the conversations and messages collections of the Firestore database. Includes a single default conversation.
- Dashboard: Displays a welcome message and links to the other tools, and the latest saved strategies/ideas/pitch decks/guides for the logged in user.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2), inspired by the vast potential of new business ideas.
- Background color: Light grey (#F0F2F5), to ensure comfortable reading.
- Accent color: A complementary purple (#9C27B0) to guide the eye and highlight calls to action.
- Body and headline font: 'Inter', a grotesque sans-serif with a neutral, objective look.
- Clean, minimalist icons to represent each tool and action.
- A clear, intuitive layout that guides users through the business strategy generation process.
- Subtle transitions and animations to provide feedback and enhance user engagement.
